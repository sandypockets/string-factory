# `kebabCase()`
Returns a string in kebab-case.

## Usage
```javascript
import { kebabCase } from 'string-factory';

kebabCase(string);
```

## Prop types
| prop     | type     | required | default   |
|----------|----------|----------|-----------|
| `string` | `String` | yes      | `null`    |

## Examples
```javascript
kebabCase('some new file name') // some-new-file-name
```
