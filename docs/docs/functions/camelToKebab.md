# `camelToKebab()`
Returns a kebab-cased string from a camelCased string.

## Usage
```javascript
import { camelToKebab } from 'string-factory';

camelToKebab(string);
```

## Prop types
| prop      | type     | required | default   |
|-----------|----------|----------|-----------|
| `string`  | `String` | yes      | `null`    |

## Examples
```javascript
camelToKebab('someCamelCase') // some-camel-case
```
