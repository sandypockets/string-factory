# `titleCase()`
Converts a string to Title Case.

## Usage
```javascript
import { titleCase } from 'string-factory';

titleCase(string);
```

## Prop types
| prop     | type     | required | default   |
|----------|----------|----------|-----------|
| `string` | `String` | yes      | `null`    |

## Examples
```javascript
titleCase('this is the article title') // This Is The Article Title
```
