# `pascalCase()`
Converts a sentence case string to a PascalCase string.

## Usage
```javascript
import { pascalCase } from 'string-factory';

pascalCase(string);
```

## Prop types
| prop     | type     | required | default   |
|----------|----------|----------|-----------|
| `string` | `String` | yes      | `null`    |

## Examples
```javascript
pascalCase('hello world') // "HelloWorld"
```
