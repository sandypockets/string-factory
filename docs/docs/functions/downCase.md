# `downCase()`
Returns a string with all characters converted to lowercase.

## Usage
```javascript
import { downCase } from 'string-factory';

downCase(string);
```

## Prop types
| prop     | type     | required | default   |
|----------|----------|----------|-----------|
| `string` | `String` | yes      | `null`    |

## Examples
```javascript
downCase('Hello World') // "hello world"
```
