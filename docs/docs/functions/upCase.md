# `upCase()`
Returns a string with all characters in the string converted to uppercase.

## Usage
```javascript
import { upCase } from 'string-factory';

upCase();
```

## Prop types
| prop       | type   | required | default   |
|------------|--------|----------|-----------|
| `prop`     | `type` | yes      | `null`    |

## Examples
```javascript
upCase('Hello World') // HELLO WORLD
```
