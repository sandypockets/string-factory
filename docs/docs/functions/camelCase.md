# `camelCase()`
Returns a string in camelCase.

## Usage
```javascript
import { camelCase } from 'string-factory';

camelCase(string);
```

## Prop types
| prop     | type     | required | default   |
|----------|----------|----------|-----------|
| `string` | `String` | yes      | `null`    |

## Examples
```javascript
camelCase("great for automating file name creation") // greatForAutomatingFileNameCreation
```
