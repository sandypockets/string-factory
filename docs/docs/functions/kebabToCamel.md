# `kebabToCamel()`
Returns a string with kebab-case words converted to camelCase.

## Usage
```javascript
import { kebabToCamel } from 'string-factory';

kebabToCamel(string);
```

## Prop types
| prop       | type   | required | default   |
|------------|--------|----------|-----------|
| `prop`     | `type` | yes      | `null`    |

## Examples
```javascript
kebabToCamel("kebab-to-camel-case") // kebabToCamelCase
```
